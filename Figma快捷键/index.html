Figma快捷键

本文最后更新于 2024-12-11T09:36:28+08:00

▶
本地变量
  • number : 数值，适用于尺寸、圆角、自动布局等属性
  • color : 颜色填充
  • string : 字符串，适用于文本图层、变体属性
  • boolean : 布尔，适用于开关切换

快捷键

工具

  • 移动：v
  • 缩放：K
  • 抓手：H
  • 评论：C
  • 文本：T
  • 开发模式：shift + d
  • 工具：command + k
  • 画框：F
  • 分区：shift + s
  • 切片：S

形状

  • 钢笔：P

  • 铅笔：shift + p

  • 矩形：R

  • 直线：L

  • 箭头：shift + L

  • 椭圆：O

    • 正圆：shift + O

  • 图片/视频：shift + command + k

视图

  • 搜索文件：command + /

  • 切换 icon：command + option

  • 自动布局：shift + A

  • figma 重命名：command + R

  • 锁定 / 解锁图层：command + shift + L

  • 快速折叠侧边栏：option + L

想要设计稿拉长距离，内容不变： 只需要在容器上按住command 键进行拉伸即可
