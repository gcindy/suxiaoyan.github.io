分辨率：单位(ppi)

  • 像素

  • 分辨率：一立方厘米里面有多少像素(即一立方厘米里面有多少长*多少宽的格子)
    • 尺寸越大，分辨率越小
    • 分辨率越大，像素大小越小
    • 分辨率越大，画面越细腻

图片格式

  • jpg / jpeg：做兼职或者平时预览的图片，只有一个图层。注意：发给客户不要发高清的，除非支付完设计费用；品质越高画面越细腻；兼职记得 🏠 水印；文件大小根据实际情况
    • 适用范围
      • 修图
      • 海报
    • 特点
      • 内存小
      • 画质高
      • 压缩速度快
  • gif：动态图
    • 适用范围
      • 表情包
      • 电商动图
  • Png：无背景清晰原图（一般是素材文件格式、免抠图；可以是透明底）
  • psd：ps 制作可修改源文件(注意：一般不要发源文件给对方；版权)
    • 特点
      • 方便修改

保存方式

  • 存储副本：command + shift + s
  • 导出：option + command + shift + w
  • web 所用格式(旧版)：option + command + shift + s

颜色模式

  • 位图模式：只使用黑白两种颜色表示像素

  • 位图：点阵图像 / 栅格图像，是由称作像素（图片元素）的单个点组成的

  • 灰度模式：只使用黑白灰三种颜色表示像素

  • RGB 模式：电子屏幕计算颜色（三原色： 红、绿、蓝）==> 适用于物体本身就能发光的物体。  例如：电脑、手机、电视机

  • CMYK 模式：油墨印刷计算颜色（青蓝色、品红色、黄色、黑色）==> 由太阳光照射反射出的颜色例如：打印的海报、印刷的书本

  • LAB 模 式：人眼所看到的所有色彩的色彩模式

快捷键

下列快捷键都是我使用系统默认&自定义的 - 标尺：command + R

  • 矩形：R

  • 圆形：O

  • 三角形：S

  • 多边形：D

  • 直线：L

  • 钢笔：P

  • 文本工具：T

  • 吸取颜色：I

  • 画笔工具：B

  • 橡皮擦工具：E

  • 裁剪工具：C

  • 自由变换：command + T

  • 颜色填充

    • 切换前景色：option + delete
    • 切背景色：command + delete
    • 切换前景色 / 背景色：X / Shift + X

  • 选取：M

    用于抠图、修色、调色、合成等

    • 添加到选取：shift

    • 从选取减去：option

    • 与选取交叉：option + Shift

    • 取消选取：command + D

    • 反选：command + Shift + i

    • 复制选区：command + J
    • 边界
    • 扩展
    • 收缩
    • 羽化：先羽化再选取
    • 存储选区 & 载入选区
    • 变换选区

  • 魔法棒：Y

  • 对象选择工具：W

  • 快速选择工具：W

  • 图框工具：K

  • 修复工具：J

    • 污点修复画笔
    • 修复工具
    • 修补工具
    • 内容识别

  • 渐变工具：G

    • 线性渐变
    • 径向渐变
    • 角度渐变
    • 对称渐变
    • 菱形渐变

  • 模糊工具

    • 模糊：把图像变模糊
    • 锐化：把模糊的图像变清晰
    • 涂抹：类似于液化

  • 减淡工具：U

    • 减淡 / 加深：把画面变白 / 深

    • 海绵

      • 去色：吸取有颜色的部分，让颜色变为黑 ==> 降低饱和度
      • 加色：吸取有颜色的部分，让颜色变为白 ==> 提亮饱和度

  • 五大图层

    • 新建图层：command + shift + N

    • 合并图层：command + E

    • 盖印(可见)图层（合并所有可见图层并生成新的图层）：command + option + shift + E

    • 调整图层

      • 上移：command + ]
      • 下移：command + [
      • 置顶：command + shift + ]
      • 置底：command + shift + [

    • 图层编组：command + G

    • 取消图层编组：command + shift + G

    • 图层基础操作

      • 锁定所有：command + /
      • 位置锁定
      • 图像锁定
      • 透明图层锁定
      • 不透明度 & 填充

    • 智能图层:起到一个保护图层的作用

    • 文字图层

    • 形状图层

  • 调整图层

  • 四大蒙版

    • 图层蒙版
    • 剪贴蒙版：command + option + G
    • 快速蒙版：command + option + G
    • 矢量蒙版：command + option + G

  • 滤镜库
